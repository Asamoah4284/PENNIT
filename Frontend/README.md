# Pennit

A reading and writing platform for African writers and readers. Discover poems, short stories, and novels. Publish your work. Find your audience.

## Tech stack

- **Frontend**: Vite, React, React Router, Tailwind CSS
- **Backend**: Not yet implemented (Node.js planned). Current build uses mock data and localStorage for auth.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Project structure

- `src/pages/` – Route pages (Home, Login, Signup, Reader, Reading, Author, Dashboard)
- `src/components/` – Layout, Header, Footer
- `src/lib/` – Mock auth (`auth.js`), API stubs (`api.js`)
- `src/data/` – Mock works and authors (`mock.js`)

## Pre-launch

Launching February–March 2026. Monetization (subscription for readers, payouts for writers) will be built in but not activated at launch.
